# AchievementAttributes

Used in the API.StatsAndAchievements.Client.GetAchievementDisplayAttribute(System.String, Heathen.SteamworksIntegration.AchievementAttributes) Method to read a specific attribute

## Values

### desc

Get the description of the achievement

### hidden

Return a value that indicates if the achievement is hidden from users

### name

Get the name of the achievement
