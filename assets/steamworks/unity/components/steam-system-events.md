## Introduction

Exposes Steam initialization events to the Unity Inspector

## Definition

```csharp

```

## Events

### evtSteamInitalized

Occurs when the Steam API is initalized

### evtSteamInitalizationError

Occurs when there is an error initalizing the Steam API
